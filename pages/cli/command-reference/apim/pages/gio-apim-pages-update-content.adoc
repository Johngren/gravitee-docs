== Description

Updates API or Portal documentation

== Usage

[subs="+macros"]
----
gio apim pages update-content <<Options,options>>
----

== Options

[cols="3", options="header"]
|===
|Option
|Description
|Default

|`-p`, `--page`
|Page ID
|

|`--api`
|API ID
|

|`-f`, `--file`
|Document file, can be a path or URL (required)
|

|===
