== Description

Runs a series of tests to verify that the API specification is correctly formed

== Usage

[subs="+macros"]
----
gio apim apis spec lint <<options>>
----

== Options

[cols="3", options="header"]
|===
|Option
|Description
|Default

|`--api`
|API ID (required)
|

|`-f`, `--file`
|Path of Swagger 2.0 or OAS 3.0 specification file (required)
|

|===
